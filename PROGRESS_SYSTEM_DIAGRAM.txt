┌──────────────────────────────────────────────────────────────────────────────┐
│                              PROGRESS FORMULAS                               │
└──────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ CHAPTER PROGRESS                                                            │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   Formula: (completedLessons / totalLessons) × 100                         │
│                                                                             │
│   Example:                                                                  │
│   ┌──────────────────────────────────────────────────────────────┐        │
│   │ Chapter: "Introduction to Grammar"                           │        │
│   │ Total Lessons: 5                                             │        │
│   │ Completed: 3                                                 │        │
│   │                                                               │        │
│   │ Progress = (3 / 5) × 100 = 60%                              │        │
│   └──────────────────────────────────────────────────────────────┘        │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ COURSE PROGRESS                                                             │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   Formula: (completedLessons / totalLessons) × 100                         │
│                                                                             │
│   Example:                                                                  │
│   ┌──────────────────────────────────────────────────────────────┐        │
│   │ Course: "English Grammar Fundamentals"                       │        │
│   │ Total Lessons: 20                                            │        │
│   │ Completed: 15                                                │        │
│   │                                                               │        │
│   │ Progress = (15 / 20) × 100 = 75%                            │        │
│   └──────────────────────────────────────────────────────────────┘        │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ PACK PROGRESS (WEIGHTED - CORRECT)                                          │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   Formula: (Σ completedLessons / Σ totalLessons) × 100                    │
│                                                                             │
│   Example:                                                                  │
│   ┌──────────────────────────────────────────────────────────────┐        │
│   │ Pack: "Complete English Course Bundle"                       │        │
│   │                                                               │        │
│   │ Course 1: Grammar Basics                                     │        │
│   │   • Total: 10 lessons                                        │        │
│   │   • Completed: 10 lessons (100%)                             │        │
│   │                                                               │        │
│   │ Course 2: Vocabulary Building                                │        │
│   │   • Total: 10 lessons                                        │        │
│   │   • Completed: 5 lessons (50%)                               │        │
│   │                                                               │        │
│   │ Course 3: Pronunciation                                      │        │
│   │   • Total: 5 lessons                                         │        │
│   │   • Completed: 0 lessons (0%)                                │        │
│   │                                                               │        │
│   │ ─────────────────────────────────────────────────────────── │        │
│   │                                                               │        │
│   │ OLD (Average - WRONG):                                       │        │
│   │   (100% + 50% + 0%) / 3 = 50%                               │        │
│   │   ❌ Treats all courses equally                              │        │
│   │                                                               │        │
│   │ NEW (Weighted - CORRECT):                                    │        │
│   │   (10 + 5 + 0) / (10 + 10 + 5) × 100                        │        │
│   │   = 15 / 25 × 100                                            │        │
│   │   = 60%                                                       │        │
│   │   ✅ Accounts for different lesson counts                    │        │
│   └──────────────────────────────────────────────────────────────┘        │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
